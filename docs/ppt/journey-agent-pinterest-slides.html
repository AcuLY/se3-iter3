执行链

执行上下文与改动全链路可追溯

Context 读取上下文 当前行程、已导入 Skill、saved memories 进入本轮输入
Tool Call 调用规划工具 新增、替换、调整或只读查询都用参数表达
Patch 写入行程对象 工具结果先进入行程状态，不直接散落在聊天文本里
Trace 记录运行轨迹 工具调用、上下文处理和异常状态形成可查记录
Diff 形成本轮改动 用户能看到Agent 本轮改了哪些画布内容
用设计化执行链表达工具调用记录，不依赖小截图
